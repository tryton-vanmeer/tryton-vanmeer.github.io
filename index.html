Titanium-Server.local

Dashboard for local media server.

Formats and displays information from the APIs of applications that run on the server. Also provides links to the applications subdomain.

Backend is written in Python and uses the Flask framework. API calls are made manually with `requests` or using a application specific module.

The repo also includes files for deployment:

  • An NGINX conf file. Passes a uWSGI socket and defines subdomains that proxy pass to the ports of the hosted applications
  • An INI to pass to uWSGI when starting the service
  • A Systemd service file
